# Ecommerce Power BI Capstone

## Project Overview
This end-to-end ecommerce analytics project focused on transforming raw transactional data into actionable business insights using Python and Power BI.

The project involved cleaning multiple raw datasets, merging them into a final analytical dataset, engineering new features, and building an executive-style dashboard to analyze ecommerce sales performance.

---

## Business Problem
The business needed better visibility into:

- Total sales performance
- Order volume
- Average order value
- Shipping costs
- Top-performing cities
- Top-performing states
- Product category revenue
- Monthly sales trends
- Order fulfillment status

## Data Cleaning Process

### Combined multiple datasets:
- Orders
- Order Items
- Payments
- Products
- Customers

### Tasks completed:
- Removed duplicate order records
- Handled missing values
- Merged datasets using common keys
- Created total order value metric
- Created purchase year/month features
- Standardized final dataset for dashboard reporting

---

## Dashboard KPIs

- Total Revenue: **1.49M**
- Total Orders: **89,316**
- Average Order Value: **16.72K**
- Average Shipping Cost: **1.21K**

---

## Dashboard Insights

- São Paulo generated the highest number of orders
- Top cities contributed most of total revenue
- Product categories showed strong concentration in specific segments
- Most orders were successfully delivered
- Monthly sales trends revealed seasonal fluctuations
